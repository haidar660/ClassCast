ClassCast Frame Extraction Package
===================================

Version: 1.0
Date: 2025-11-26

WHAT'S INCLUDED
---------------

Core Files:
  extract_frames.py         - Main frame extraction script
  requirements.txt          - Python dependencies
  test_installation.py      - Installation verification script
  example_usage.py          - Python integration examples

Documentation:
  README.md                 - Complete documentation
  QUICKSTART.md            - Quick start guide
  SETUP.md                 - Detailed setup instructions
  PACKAGE_INFO.txt         - This file

QUICK START
-----------

1. Install dependencies:
   pip install -r requirements.txt

2. Test installation:
   python test_installation.py

3. Extract frames:
   python extract_frames.py your_video.mp4

SYSTEM REQUIREMENTS
-------------------

- Python 3.8 or higher
- opencv-python package
- 2GB+ RAM
- 500MB+ disk space per 10-minute video

FEATURES
--------

✓ Extract frames at configurable intervals
✓ Time range selection
✓ Adjustable quality settings
✓ Automatic metadata generation
✓ Cross-platform support (Windows, macOS, Linux)
✓ Progress tracking
✓ JSON metadata output
✓ Integration with ClassCast pipeline

GETTING HELP
------------

1. Read QUICKSTART.md for immediate usage
2. Read README.md for comprehensive documentation
3. Run: python extract_frames.py --help
4. Check example_usage.py for code examples

INTEGRATION
-----------

This package integrates with the ClassCast video-to-podcast pipeline:

  Frame Extraction → OCR → ASR → Fusion → TTS → Podcast

The output format is compatible with the ClassCast data models:
- FrameInfo objects
- Timestamp synchronization
- JSON metadata for downstream processing
